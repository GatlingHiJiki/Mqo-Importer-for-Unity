Mqo-Importer-for-Unity
======================
Metasequoia　モデルデータ・フォーマット（.mqo）の読み込みを目的としたインポーター.

###使用するには
===============

プロジェクト追加後、メニューバーにImporterタブが出現.<br\/>
そのタブのMQO Loaderを選んでください.
ウィンドウが表示されます.<br\/>
そのウィンドウの空欄に、．mqoファイルをD&Dし、
Convertボタンをクリックすることでprefab化されます.<br\/>
